# NAME

Catmandu::Z3950 - Catmandu module for working with Z3950 data

# VERSION

Version 0.04

# SYNOPSIS

    $ catmandu convert Z3950 --host z3950.loc.gov --port 7090 --databaseName Voyager --query "(title = dinosaur)"
